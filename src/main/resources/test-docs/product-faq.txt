产品FAQ文档
产品部 2025年Q4版本

Q1: 如何开始使用产品？

A: 注册账号后，系统会自动为您创建一个默认工作空间。
完成以下步骤即可开始使用：
1. 在个人设置中完善账号信息
2. 进入控制台，选择您需要的功能模块
3. 参考《快速入门指南》完成初始化配置

Q2: 免费版和付费版有什么区别？

A: 免费版功能限制如下：
- AI问答：每日100次
- 知识库文档：最多50个
- 存储空间：1GB
- 不支持API接入

付费版（专业版）：
- AI问答：每日10000次
- 知识库文档：无限制
- 存储空间：100GB
- 支持API接入，每日1000次API调用
- 优先客服支持

Q3: 如何申请API访问权限？

A: 申请API访问权限步骤：
1. 登录控制台，进入「开发者中心」
2. 点击「创建API Key」，填写用途说明
3. API Key立即生效，免费配额：每日1000次调用
4. 如需更高配额，可在「套餐管理」中升级付费套餐

API调用频率限制：
- 免费版：每分钟10次，每日1000次
- 付费版：每分钟100次，每日10000次

Q4: 数据安全如何保障？

A: 我们采用多层次的数据安全保障机制：
1. 传输安全：全程HTTPS加密，防止中间人攻击
2. 存储安全：数据加密存储，密钥定期轮换
3. 访问控制：基于角色的权限管理（RBAC），最小权限原则
4. 数据隔离：多租户架构，不同用户数据完全隔离
5. 备份机制：每日自动备份，保留30天，支持时间点恢复

Q5: 知识库支持哪些格式的文档？

A: 目前支持以下格式：
- PDF（包含扫描件，支持OCR识别）
- Word文档（.docx格式）
- Markdown文件（.md格式）
- 纯文本文件（.txt格式）

单个文件大小限制：50MB
建议文档在100页以内，过大的文档建议拆分上传

Q6: AI回答不准确怎么办？

A: 如果发现AI回答不准确，可以：
1. 点击答案下方的「不好用」按钮，留下反馈
2. 检查知识库文档是否包含该问题的答案
3. 尝试换个方式提问，更具体的问题通常效果更好
4. 如果知识库文档有更新，记得重新上传最新版本

我们会根据用户反馈持续优化AI模型，您的反馈非常重要。
